# Market Intelligence Dashboard

A full-stack Market Intelligence Dashboard built using Spring Boot, React, PostgreSQL, and Google Cloud Run.

## Features

- Interactive Dashboard
- Business Analytics
- Global Superstore Dataset
- Search Products
- Category Analytics
- Regional Analytics
- Market Trends
- Interactive Charts
- REST APIs
- Swagger Documentation

## Tech Stack

### Backend
- Spring Boot
- Spring Data JPA
- Flyway
- PostgreSQL
- OpenCSV

### Frontend
- React
- Vite
- Axios
- Recharts

### Deployment
- Google Cloud Run
- Vercel
- Supabase PostgreSQL

## Dataset

Global Superstore Dataset
